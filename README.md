# agent-knowledge-board-skill

Agent Knowledge Board is a reusable Codex skill for turning agent-collected messages, feeds, newsletters, and research outputs into readable knowledge cards.

Use it when you want:

- V2 hand-journal daily-board email digests at 720px width
- ljg-card-style multi-card PNG summaries for WeChat forwarding
- categorized daily boards from multiple agents or channels
- optional links to prior notes in a local knowledge base
- channel-ready card plans for email, web, Feishu/Lark, DingTalk, WeCom, WeChat image cards, and Markdown fallback

Invoke it as:

```text
$agent-knowledge-board
```

The main skill file is `SKILL.md`.
